# 덧셈식 출력하기

### **문제 설명**

두 정수 `a`, `b`가 주어질 때 다음과 같은 형태의 계산식을 출력하는 코드를 작성해 보세요.

`a + b = c`

---

### 제한사항

- 1 ≤ `a`, `b` ≤ 100

---

### 입출력 예

입력 #1

`4 5`

출력 #1

`4 + 5 = 9`

### 소스 코드

```python
a, b = map(int, input().split(' '))
print(a, "+", b, "=", a+b)
```

### 느낀 점

```python
a, b = map(int, input().strip().split(' '))
c = a + b
print('{} + {} = {}'.format(a, b, c))
```

이렇게 format() 함수를 사용한 포맷팅도 좋은 것 같다
